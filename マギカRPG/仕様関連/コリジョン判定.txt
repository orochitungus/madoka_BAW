
加藤 政樹 僕は「オンラインゲームのしくみ」という本の「どきどきがんもどき」というゲームを作ったときは、次のようにしていました。

１．OnCollisionStay() でヒットしたコリジョンをリストに追加
２．Update() で リストに敵が入っていたら、攻撃モーションの再生やダメージ付与を行う
３．２．で 処理の終わったヒット相手をリストから取り除く
４．FixedUpdate() でリストをクリアー（空っぽ）する

　このとき作成したゲームでは、マウスで移動するキャラが敵に接触すると自動的に近接攻撃を行います。トリガーではなくコリジョンを使用していますが、島国さんのおっしゃっている仕様に近いのではないかと思います。１．でリストに追加する際、重複しないようなチェックを入れればよりナイスですね。

　ご参考まで。

　(*) いつも島国さんのサイト、楽しく読ませてもらっています！

// コリジョンにヒットしている間中よばれるメソッド.
void OnCollisionStay(Collision other)
 {
 switch(other.gameObject.tag) {

 case "Item":
 case "Enemy":
 case "EnemyLair":
 case "Boss":
 {
 CollisionResult result = new CollisionResult();

 result.object0 = this.gameObject;
 result.object1 = other.gameObject;
 result.is_trigger = false;

 this.control.collision_results.Add(result);
 }
 break;
 }
 }

 void Update()
 {
 foreach(var result in this.collision_results) {

 GameObject other = result.object1;

 switch(other.tag) {

 case "Enemy":
 case "EnemyLair":
 case "Boss":
 {
 // 敵と接触したら近接攻撃.
 }
break;
 }
 }
 }

 void FixedUpdate()
 {
 this.collision_results.Clear();
 }
